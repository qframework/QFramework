Clients directory. Consists of various client implementations.

AndroidQ  - android implementation, works with opengl es 1.0
	HellouQAnd - Hellou world eclipse android project 
	QAnd - android Q framework eclipse project
	tutorials - q framework tutorials
	
Html5Q - Html5 implementation, works with WebGl
	HellouQ - Hellou world application
	tutorials - q framework tutorials

iPhoneQ - IOS, implementation. opengl es 1.0, iPad, iPod, iPhone
	HellouQ - XCode Hellou project
	Q - Q framework ios implementation source code, and scripts.
	tutorials - q framework tutorials XCode project
	
JavaQGl -  Java port, applet and application, JogAmp for openGl support.
	HellouQ - Eclipse Hellou project
	Q - Q framework eclipse project
	tutorials - Eclipse q framework tutorials project
